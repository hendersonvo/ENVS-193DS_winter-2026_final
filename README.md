# ENVS 193DS Winter 2026 Final
Final project for ENVS 193DS Winter 2026

## General information

This repository contains the contents for the Final assignment, including descriptions of research writing critiques, code for data visualizations, code and interpretation for data analyses, and a revisiting of personal data to create an affective visualization.

This is my personal repository that I created. I am the sole contributor.

## Data and file overview

All code files are in the 'code' folder. There is an .Rmd file that contains the responses, plots, and images to all questions from the Final. This includes code to create visualizations to demonstrate reproducibility, exploratory data analysis, diagnostic testing, and statistical assumption checking. The code also includes statistical tests to analyze a Mountain Plover research article as well as personal data.
